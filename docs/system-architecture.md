## 五、重複問題總結

### 門檻重複（同一套規則，只是數字不同）
- **v1 / v2 / 保守** → 同樣8個規則組，只是量比(1.3/1.8/2.0)和分數(4/4/6)不同
- **v3 / 台股v3 / 陸股v3** → 同樣全部規則，門檻差距極小(量比差0.1~0.2)
- **五步驟 vs 回測驗證版** → 門檻幾乎一樣，回測版多了lin-sop+bollinger 2個規則組

### 真正不同的策略思路只有 4 個
1. **朱老師體系** — v1/v2/保守/五步驟/回測驗證版都是
2. **多因子全規則** — v3系列，150+條全開
3. **走圖SOP** — 只用林穎8條規則
4. **大師共識** — 只看多師一致+葛蘭碧

### 統計
- **總規則數**：~150 條
- **總規則組**：16 組
- **總策略**：10 個（真正獨特的只有4個）
