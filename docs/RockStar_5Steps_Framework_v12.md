### 書本原文（寶典 p.652）

| 走勢 | 內容 | 操作 |
|---|---|---|
| 多頭 | **頭頭高、底底高** | 順勢做多 |
| 空頭 | **頭頭低、底底低** | 順勢做空 |
| 盤整 | 不是多頭也不是空頭 | 退出觀察，等突破或跌破 |

### 規則細節

- **判定基礎**：用「已確認 pivot」（不用 provisional），最近兩個高點 + 兩個低點皆嚴格交替
- **嚴禁加容差**（記憶 0421）—— 容差會偏袒多頭、偽造底底高
- 即時翻多：今日 close 突破最近確認頭時，可即時轉多頭判定
- 季線（MA60）若仍下彎且在股價上方，視為**壓力扣分**，不直接擋多頭判定（寶典 p.59 註）

- `lib/analysis/detectTrend.ts`
- `lib/analysis/findPivots.ts`（pivot 嚴格交替；交界日雙重計算）

### 書本原文（寶典 p.653）

| 走勢 | 均線架構 |
|---|---|
| 多頭 | **最少 3 線多排**，向上，股價在月線之上 |
| 空頭 | 最少 3 線空排，向下，股價在月線之下 |
| 盤整 | 雜亂無章，退出觀察等正常排列 |

### 規則細節

- **短線多頭排列定義**：MA5 > MA10 > MA20，且三線斜率向上
- **季線壓力**：MA60 仍在股價上方且下彎時，視為扣分項（寶典 p.59）
- **葛蘭畢 8 法則**（買進 4 + 賣出 4，鏡像）：
  1. 黃金交叉（均線轉平緩或上揚 + 股價向上突破）
  2. 回跌支撐後再買（不破前波底）
  3. 假跌破買上漲（3-5 日彈回均線之上）
  4. 跌深搶反彈（乖離過大，逆勢短線）
- **三條均線戰法 SOP**（抓線圖 p.261-265）：
  - 用 MA3 + MA10 + MA24（**不是** MA5/10/20）
  - 進場：MA3+MA10 黃金交叉 + 股價在 MA24 之上 + MA24 向上 + 站上 MA3
  - 出場：MA3+MA10 死亡交叉 + 跌破 MA3
  - 停損：MA10

|---|---|

| 共通 | 起漲量 / 壓力量 / 高檔量、量價背離、成交量定義看日後價漲跌 |

### 規則細節

- **底部打底量**：底部出現大量代表有人買進（寶典 p.59）

- **量價背離**：寶典 Part 11-3 第 1-2 口訣

- **關前放量警示**：寶典 Part 11-3 第 11 口訣 — 關前放大量、股價不漲要回檔
- **CN 量單位**：EastMoney/Tencent 回傳「手」需 × 100 換股（記憶 0422 修復）

```typescript
function isRedKBodyValid(kbar, prevClose, market, symbol): boolean {
  // 一般情況：實體 ≥ 2%
  const bodyPct = (kbar.close - kbar.open) / kbar.open;
  if (bodyPct >= 0.02) return true;

  // 漲停例外（議題 64）

  const gapPct = (kbar.open - prevClose) / prevClose;

  return false;
}
```
