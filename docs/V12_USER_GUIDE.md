## 2. Step 0 大盤過濾（進場做多最高前提）

掃描面板**最頂端 banner**（綠/紅/黃條）：
- 🟢 多頭：可正常做多
- 🔴 空頭：**全市場停止做多**（書本：站不上月線不進場）
- 🟡 盤整：可選擇性做多，謹慎

## 3. Step 1 預選池（A tab）

過六條件 + 戒律 + 淘汰法 → 進預選池。**A tab 不是進場訊號**，只是合格池。

## 4. Step 2 進場（B-Q tabs）

切到任一個 B-Q tab，看當天該訊號命中的個股。卡片上：

- 卡片顯示策略名稱、price、changePct、跨策略命中徽章
- **N 型態確認**多顯示：`圓弧底 85% →12.7%` （型態名 / 達成率 / 目標漲幅）
- **F V 反轉**多顯示：`🔒 507.44`（V 底反彈起點）
- **K/D**多顯示：`⏳ 2 天` / `✓ 確認` / `✗ 撤銷`（三天驗證）
- 警示徽章：`末升段` / `季壓 N` / `爆量` / `KD↓`

## 5. LockWatch 鎖股觀察名單（F/N 專用）

掃描面板上方第二區塊。F V 反轉 + N 型態確認觸發時自動寫入觀察名單。

每筆顯示：
```
[F][2330] @1085 →1110 95% [可進場 +15d]  [+] [🛒] [✕]
```

按鈕：
- **🛒 進場**：跳到 `/portfolio?prefill=…` 帶入 v12 欄位
- **+ 加自選**：加入 watchlist
- **✕ 移除**：手動移除（議題 17 純書本不限期）

## 6. Step 3-5 持倉訊號（在持倉每筆下方）

任一持倉下方有色帶：
```
▶ V12 Step 3-5 訊號 · B · 短線 · 損益 +4.09%
```

點開展開三段：

### Step 3 停損
```
📉 Step 3 停損                           2030.00 (-11.4%)
   K 線三段式 / pivot low / 黑 K low（依字母）
   K 線三段式 2030.00 · 10% 絕對下限 1980 · 末升段 trailing 啟用
```

如果觸發 5 條絕對停損之一：
```
🚪 強制出場：⑥-1 跌破盤整區 / ⑥-2 多頭翻空頭 / ⑥-4 跌幅 >10% / ⑥-5 跌破 V 底
```

### Step 4 操作
```
🎯 Step 4 操作                           MA5 @2275
   ⚠️ 乖離 ≥15% — Step 5 ② 自動切 MA5 跟隨（不直接停利）
   ⚠️ Step 4 ① 收盤跌破前一日最低
   操作模式：[短線] [長線] [波段]                  [⬆ 升級長線]
```

### Step 5 停利
```
💰 Step 5 停利                           輕微獲利
   ℹ️ 獲利達 10% — 可考慮升級長線
   🛡️ 寶典 #5/#6 進階紀律已啟用：<10% 跌破 MA5 續抱、≥10% 才停利
   📛 末升段觸發（≥100% 起漲）— 切 trailing 3% (recentHigh × 0.97 = 980.45)
```

### 紀律切換按鈕
```
手動切換：[進階紀律 ✓] [末升段 ✓]                 [🔄 重新計算]
```

## 7. 走圖頁 v12

切到任一個股票走圖時：

### 訊號 tab（v12 14 軌訊號）
最上方新區塊「**v12 14 軌訊號**」即時跑 M/N/O/P/Q 5 個 detector，命中時顯示三軌色票 + 詳細說明。

### K 線歷史 markers
K 線上自動標出過去 90 天觸發過的 v12 訊號日期：
- M / N / O / P / Q / F 字母標記在對應 K 棒下方
- N（三重底/頭肩底）標記較粗（strength=2）

## 8. Re-entry 候選名單

掃描面板下方「Re-entry」區塊（議題 28）。賣出後系統會自動找符合再進場條件的股票（過去 N 天賣出 + 今天再次 trigger）。

## 9. 自選股 conditions

點任一自選股 → 顯示該股當下命中哪些策略字母（A-Q）。

## 進場 → 持倉 → 賣出完整流程

```
1. 在掃描頁 N tab 看到 4966.TWO 觸發圓弧底（85% 達成率）
   → 卡片顯示 圓弧底 85% →573.35
2. LockWatch 自動加入 4966.TWO（F/N 都會走觀察名單）
   → 等個股 detectTrend 翻多 + 大盤 Step 0 過 → 升級 entry-signal
3. 點 LockWatch 的 🛒 → 跳到 /portfolio
   → form 自動填 symbol=4966、triggerSignal=N、price=507.44
4. 填張數 + 確認新增 → 自動 markLockWatchPurchased
   → LockWatch 該筆 stage 切 'purchased'
5. 回掃描頁 → 持倉列表多 4966
   → 下方色帶顯示 Step 3-5 訊號（停損/操作/停利）
6. 每天看色帶顏色：
   - 🟢 綠：獲利 ≥10%
   - 🟡 黃：虧損中
   - 🔴 紅：出場訊號（強制停損 / MA 跌破 / K 棒反轉）
7. 達 10% → 點「⬆ 升級長線」按鈕，全部訊號統一切 MA20
8. 達 100% → 點「末升段」按鈕，切 trailing 3%
9. 觸發出場 → 賣出
10. 看 Re-entry 面板找下一個進場候選
```

## API 概覽

| 端點 | 用途 |
|---|---|
| `/api/scanner/results?market=X&direction=long&date=Y&mtf=Q` | 取某天某策略結果 |
| `/api/lockwatch?market=X` | 取最新 LockWatch snapshot |
| `/api/lockwatch/remove` (POST) | 手動移除 |
| `/api/lockwatch/mark-purchased` (POST) | 標已買進 |
| `/api/cron/update-lockwatch?market=X` | 每日盤後狀態演進 |
| `/api/portfolio/v12-signals?symbol=...&entryPrice=...` | 單支持倉 Step 3-5 訊號 |
| `/api/scanner/reentry-candidates?market=X` | Re-entry 候選名單 |

## v12 設計原則

1. **純書本**（≥95% 規則來自朱家泓 6 本書）
2. **三軌制**（多頭/轉折/戰法 各自 SOP）
3. **單一停損方法**（每訊號用一個主方法，不取 max）
4. **10% 絕對下限**（所有訊號都套）
5. **議題鎖定**（133 議題 → docs/RockStar_5Steps_Framework_v12.md）
