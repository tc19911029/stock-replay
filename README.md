# rockstock

一個基於朱家泓老師技術分析理論的股票策略研究平台，支援歷史回放、條件掃描、嚴謹回測與研究報表。

---

## 核心功能

| 功能 | 說明 |
|------|------|
| K線歷史回放 | 逐根 K 棒回放任一股票歷史走勢，模擬真實操盤節奏 |
| 條件掃描（台股/陸股） | 批量掃描台股或陸股，篩選符合朱老師六大條件的股票 |
| 嚴謹策略回測 | 指定日期回測六大條件選股績效，進場使用隔日開盤價，含完整成本模型 |
| 研究報表 | 每次掃描/回測產生可追溯的完整紀錄，支援歷史查閱 |
| 模擬交易 | 走圖練習中模擬進出場，驗證策略執行紀律 |
| 自選股監控 | 追蹤關注個股，掃描結果一鍵加入 |
| 問老師 AI | 整合 Anthropic Claude API，對掃描結果生成自然語言分析報告 |

## 六大條件 SOP

| # | 條件 | 說明 | 通過標準 |
|---|------|------|----------|
| ① | 趨勢 | 判斷大/小趨勢是否為多頭 | 頭頭高底底高，MA5 > MA20，MA20 不下跌 |
| ② | 位置 | 股價回測均線後是否在低檔翻多 | 回測 MA10 後翻多，或 MA20 乖離率 0–12% |
| ③ | K棒 | 當日 K 棒是否為強多訊號 | 長紅 K（實體 ≥ 2%），收盤在 K 棒上半段 |
| ④ | 均線 | 均線多頭排列，股價站上 MA5 | MA5 > MA10 > MA20，股價 > MA5，MA5 向上 |
| ⑤ | 量能 | 成交量是否放量突破 | 成交量 ≥ 5 日均量 × 1.5 倍 |
| ⑥ | 指標 | 動能指標是否同步翻多 | MACD 紅柱，或 KD 黃金交叉（K > D，20 ≤ K ≤ 85） |

## 回測規則說明

- **訊號生成**：掃描日收盤後評分，當日資料不含未來資訊
- **進場**：隔日開盤價（防止未來函數）
- **出場**：固定持有 N 日 / 停損 / 停利（可組合）

### 成本模型

| 市場 | 費用項目 |
|------|---------|
| 台股 | 手續費 0.1425%（買賣均收）+ 證交稅 0.3%（賣出） |
| 陸股 | 佣金 0.03%（買賣均收，最低 5 元）+ 印花稅 0.1%（賣出）+ 過戶費 0.002%（滬市） |

### 已知限制

- 不支援漲停板無法買到的情境（隔日開盤即假設可成交）
- Yahoo Finance 歷史資料可能有缺口，部分股票回測結果不完整
- 掃描池固定（非動態成交量前 N 名），名單不隨日期更新

## 掃描池說明

| 市場 | 範圍 |
|------|------|
| 台股 | 約 600 支上市/上櫃主要股票（上市主板藍籌 + 部分上櫃科技股） |
| 陸股 | 約 600 支滬深主板股票（含 603xxx 科技股，排除創業板/科創板/ST） |

### ⚠ 掃描池偏誤說明（回測研究誠信聲明）

**目前掃描池為靜態固定名單，非動態每日成交量前 N 名。**

| 項目 | 目前實作 | 嚴格研究標準 |
|------|---------|-------------|
| 掃描池 | 靜態約 600 支（人工維護名單） | 每日收盤成交量前 500 支（動態） |
| 名單更新 | 手動更新，不隨日期變動 | 每個回測日期應取當日排名 |
| 流動性篩選 | 依靠固定名單間接控制 | 動態成交量門檻過濾 |

**可能造成的偏誤：**
- 靜態名單中可能包含特定歷史時期成交量不達標的股票
- 真正的高量強勢股若不在名單中，將被遺漏
- 回測勝率可能與「真實可操作的動態高量股」有所差異

**技術原因：**
Yahoo Finance 公開 API 不提供「歷史特定日期成交量排名」，動態掃描池需要付費資料來源（如 FinMind、Fugle）。

**正確判讀方式：**
> 回測結果代表「這批靜態股票池中，符合六大條件者的後續表現」，而非嚴格意義上「每日流動性最高 N 檔」的策略績效。

## 架構說明

```
資料層：lib/datasource/（DataProvider 介面 + Yahoo Finance 實作 + 記憶體快取）
指標層：lib/indicators.ts（MA / MACD / KD）
分析層：lib/analysis/trendAnalysis.ts（趨勢 / 位置 / 六大條件）
掃描層：lib/scanner/（MarketScanner 抽象類 + TW / CN 實作）
回測層：lib/backtest/（BacktestEngine + CostModel + ForwardAnalyzer）
狀態層：store/（Zustand + 本地持久化）
UI 層：app/ + components/
```

## 安裝與啟動

```bash
git clone <repository-url>
cd rockstock
npm install
cp .env.local.example .env.local
# 填入 ANTHROPIC_API_KEY（選填，問老師 AI 功能需要）
npm run dev
```

開啟 [http://localhost:3001](http://localhost:3001)

## 已知限制

- 資料來源僅 Yahoo Finance，部分股票可能缺資料或有延遲
- **掃描池為靜態固定名單，非動態依成交量排名**（詳見上方偏誤說明）
- 不支援漲停板無法買到的情境（隔日開盤即假設可成交）
- 不支援當沖、槓桿、期貨等衍生商品策略
- 歷史回測績效僅供研究參考，不保證未來結果
